Returns a reference to the entry at specified location pos, with bounds checking.

## Declarations
```cpp
const_reference at( size_type pos ) const;
```
_(since C++23)_

## Parameters
- `pos`: position of the stacktrace entry to return

## Return value
Reference to the requested entry.
