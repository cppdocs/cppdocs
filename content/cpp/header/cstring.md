This header was originally in the C standard library as [<string.h>](/c/string/byte/).

## Notes
NULL is also defined in the following headers:
<clocale>
<ctime>
<cstddef>
<cstdio>
<cwchar>
std::size_t is also defined in the following headers:
<ctime>
<cstddef>
<cstdio>
<cuchar> (since C++17)
<cwchar>
