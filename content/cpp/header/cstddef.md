This header was originally in the C standard library as [<stddef.h>](/c/types/).

## Notes
NULL is also defined in the following headers:
<clocale>
<ctime>
<cstring>
<cstdio>
<cstdlib>
<cwchar>
std::size_t is also defined in the following headers:
<ctime>
<cstring>
<cstdio>
<cwchar>
<cuchar> (since C++17)
