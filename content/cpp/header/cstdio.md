This header was originally in the C standard library as [<stdio.h>](/c/io/).

## Notes
NULL is also defined in the following headers:
<clocale>
<ctime>
<cstddef>
<cstring>
<cwchar>
<cstdlib>
std::size_t is also defined in the following headers:
<ctime>
<cstddef>
<cstring>
<cwchar>
<cuchar> (since C++17)
<cstdlib>
