Returns a [std::text_encoding](/cpp/locale/text_encoding/) object that represents the implementation-defined encoding scheme associated with the locale *this.

## Declarations

std::text_encoding encoding() const;

## Return value
A [std::text_encoding](/cpp/locale/text_encoding/) object that represents the associated encoding scheme.
