Class template std::messages is a standard locale facet that encapsulates retrieval of strings from message catalogs, such as the ones provided by GNU [gettext](https://www.gnu.org/s/gettext/) or by POSIX [catgets](https://pubs.opengroup.org/onlinepubs/9699919799/functions/catgets.html).

## Declarations
```cpp
template< class CharT >
class messages;
```
