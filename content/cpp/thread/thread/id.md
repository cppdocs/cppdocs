The class thread::id is a lightweight, trivially copyable class that serves as a unique identifier of [std::thread](/cpp/thread/thread/) and [std::jthread](/cpp/thread/jthread/)(since C++20) objects.

## Declarations
```cpp
class thread::id;
```
_(since C++11)_
