The name cpo denotes a customization point object, which is a const [function object](/cpp/named_req/functionobject/) of a [literal](/cpp/named_req/literaltype/) [semiregular](/cpp/concepts/semiregular/) class type. For exposition purposes, the cv-unqualified version of its type is denoted as __cpo_fn.
