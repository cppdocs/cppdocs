title: "C++ named requirements: BitmaskType"
source_path: "cpp/named_req/BitmaskType"
category: "named_req"
---

Defines a type that can be used to represent a set of constant values or any combination of those values. This trait is typically implemented by integer types, [std::bitset](/cpp/utility/bitset/), or enumerations (scoped and unscoped) with additional operator overloads.
