---
title: "C++ named requirements: AccessorPolicy (since C++23)"
source_path: "cpp/named_req/AccessorPolicy"
category: "named_req"
since: "C++23"
---

AccessorPolicy defines how elements are accessed from a given data handle to such elements and an index. It is used as a policy in [std::mdspan](/cpp/container/mdspan/) to specify the access to a reference of an element using an underlying 1D index.
