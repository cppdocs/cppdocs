Swaps the contents with those of other.

## Declarations
```cpp
void swap( valarray& other );
```
_(until C++11)_

```cpp
void swap( valarray& other ) noexcept;
```
_(since C++11)_

## Parameters
- `other`: another valarray to swap the contents with

## Return value
(none)
