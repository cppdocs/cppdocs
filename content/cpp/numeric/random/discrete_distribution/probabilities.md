Obtains a [std::vector](/cpp/container/vector/)<double> containing the individual probabilities of each integer that is generated by this distribution.

## Declarations
```cpp
std::vector<double> probabilities() const;
```
_(since C++11)_

## Return value
An object of type [std::vector](/cpp/container/vector/)<double>.

## Example
```cpp
#include <iostream>
#include <random>
#include <vector>
 
int main()
{
    std::discrete_distribution<> d({40, 10, 10, 40});
    std::vector<double> p = d.probabilities();
    for (auto n : p)
        std::cout << n << ' ';
    std::cout << '\n';
}
```
