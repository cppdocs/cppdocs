Returns the n parameter the distribution was constructed with. It specifies the [degrees of freedom](https://en.wikipedia.org/wiki/Degrees_of_freedom_(statistics)) of the distribution. The default value is 1.0.

## Declarations
```cpp
RealType n() const;
```
_(since C++11)_

## Return value
The n distribution parameter.
