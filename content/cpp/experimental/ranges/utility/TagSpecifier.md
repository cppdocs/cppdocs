The concept TagSpecifier<T> is satisfied if and only if T is a tag specifier for use with [ranges::tagged](/cpp/experimental/ranges/utility/tagged/).

## Declarations
```cpp
template< class T >
concept bool TagSpecifier = /* implementation-defined */;
```
