A function is a C language construct that associates a [compound statement](/c/language/statements/#Compound_statements) (the function body) with an [identifier](/c/language/identifier/) (the function name). Every C program begins execution from the [main function](/c/language/main_function/), which either terminates, or invokes other, user-defined or library functions.
